# V2 Sprint 7：设备认证策略

## 认证方式

设备认证采用受控方案，不依赖固定共享密码：

- 一次性注册令牌
- 设备密钥对
- 设备指纹
- 请求时间戳
- Nonce 防重放
- 凭据轮换与撤销

## 注册规则

- 注册令牌只能使用一次
- 注册令牌必须有有效期
- 设备私钥不得上传服务器
- 服务端仅保存公钥、指纹或安全摘要

## 请求校验

每次设备请求都必须校验：

- 设备是否已批准
- 凭据是否有效
- 时间戳是否过期
- Nonce 是否重复
- 设备是否被禁用或撤销
- 请求签名是否匹配

## 失效策略

- 设备离线后不得继续接收任务
- 设备禁用后立即拒绝
- 设备撤销后旧凭据立即失效
