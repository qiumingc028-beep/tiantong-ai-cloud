# V2 隔离桌面策略

## 原则

隔离桌面只能用于开发和测试环境，不得连接真实生产电脑。

## 必须隔离的边界

- 用户真实 Mac 桌面
- 生产服务器桌面
- 用户 Home 目录
- SSH Key
- 浏览器真实 Cookie
- 密码管理器
- 生产 Secret

## 行为约束

1. 不打开 Terminal。
2. 不执行 Shell。
3. 不输入密码和验证码。
4. 不进行文件删除、上传、支付、下单、发布。
5. 任务结束后允许清理会话和临时资源。

## 运行模式

- Mock Computer Executor：用于默认测试。
- 隔离电脑执行器：仅限测试环境且必须具备可靠隔离能力。
- 真实电脑控制：本 Sprint 禁止。
