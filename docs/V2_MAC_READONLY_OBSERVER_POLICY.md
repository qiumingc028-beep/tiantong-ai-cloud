# V2 Sprint 7：Mac 只读观察策略

## 允许范围

只允许在授权测试设备上进行只读观察：

- 应用枚举
- 窗口枚举
- 指定窗口截图
- 屏幕状态识别
- 文字型建议生成

## 默认禁止

- 自动点击
- 自动输入
- 剪贴板读取
- 打开 Terminal
- 执行 Shell
- 访问密码管理器
- 访问钥匙串
- 访问浏览器 Cookie

## 白名单要求

应用白名单和窗口白名单都必须显式配置。
白名单为空时，必须拒绝观察。

## 敏感内容阻断

检测到以下内容时必须阻断：

- 密码
- Token
- Secret
- 私钥
- 验证码
- Keychain

阻断后不得保存敏感画面。
