# V2 电脑执行器安全策略

## 允许的动作类型

- 查看屏幕
- 获取窗口列表
- 激活允许的窗口
- 移动鼠标
- 单击
- 双击
- 滚动
- 输入普通文本
- 按允许的快捷键
- 截图
- 等待
- 返回上一步
- 取消任务

## 默认禁止

- Terminal
- iTerm
- SSH
- Shell
- 密码输入
- Token 输入
- 验证码输入
- 银行卡输入
- 身份证输入
- 删除、卸载、安装、支付、下单、发布、提交表单
- 修改系统配置
- 修改网络配置
- 访问生产系统

## 应用与窗口白名单

- 仅允许明确配置的应用和窗口模式。
- 白名单为空时直接拒绝。
- 默认阻断系统设置、钥匙串、密码管理器、远程桌面、生产运维工具。

## 截图与证据

- 只保存引用，不保存完整敏感截图。
- 截图必须脱敏。
- 截图保留周期可配置。

## 人工接管

- 默认关闭。
- 触发敏感区域时可暂停并请求人工接管。
- 接管过程必须记录状态，但不得记录敏感输入。
