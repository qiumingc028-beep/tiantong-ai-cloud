# V2 Sprint 7：Mac 测试设备受控接入架构

## 目标

本 Sprint 只接入经过授权的 Mac 测试设备，提供只读观察能力：

- 设备注册与身份认证
- 在线心跳与状态管理
- 应用与窗口枚举
- 指定窗口截图
- 屏幕状态摘要
- 下一步文字建议
- 截图证据与审计
- 人工授权与紧急断开

## 架构分层

设备测试请求必须经过以下链路：

设备注册中心 → 设备身份认证 → Agent Runtime → Skills Engine → Computer Executor → Mac Readonly Observer → 截图证据 → 审计 → Task Center 回写

任何设备观察任务都不能绕过 Agent Runtime 直接访问设备能力。

## 设备范围

当前仅支持：

- Mac 测试设备

不支持：

- 真实用户桌面
- 生产服务器桌面
- 自动点击
- 自动键盘输入
- 剪贴板读取
- Terminal / Shell

## 能力边界

Mac Observer 仅提供只读动作：

- 窗口枚举
- 白名单窗口截图
- 屏幕状态识别
- 页面状态摘要
- 文字型下一步建议

不提供：

- 鼠标控制
- 键盘输入
- 剪贴板访问
- 文件修改
- 终端操作

## 安全原则

- 设备必须经过注册、批准和心跳验证
- 设备撤销后凭据立即失效
- 请求必须防重放
- 敏感窗口直接阻断
- 非白名单窗口禁止截图
- 截图必须脱敏或拒绝保存
- 外部视觉模型默认关闭
- 生产环境默认关闭
