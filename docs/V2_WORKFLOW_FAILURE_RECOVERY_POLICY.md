# V2 工作流失败恢复策略

## 原则

工作流失败后优先停止，恢复仅限测试环境内的安全回退。

## 允许的恢复

- 返回工作流开始前的测试页面
- 清空测试文本框
- 关闭测试弹窗
- 重置本地演示页面

## 不允许的恢复

- 数据库回滚
- 真实文件回滚
- 系统设置回滚
- 账号状态回滚
- 生产环境回滚

## 失败处理

- 目标窗口变化：暂停
- 目标控件丢失：暂停
- 页面状态不符：暂停
- 敏感内容出现：终止
- 设备离线：终止
- 超时：终止
- 本地停止：立即终止

## 重试规则

- 仅允许一次有限重试。
- 只能用于短暂截图失败、短暂状态读取失败、页面加载延迟。
- 审批失败、敏感内容、禁止页面、Terminal、Shell、文件操作一律不重试。
